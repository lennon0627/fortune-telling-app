# 究極の運勢占いアプリ ✨

5種類の占術（九星気学、四柱推命、五星三心占い、カバラ数秘術、宿曜占星術）で2026年の運勢を完全鑑定するWebアプリケーションです。

## 📁 プロジェクト構成

```
fortune-telling-app/
├── main.py              # FastAPIアプリケーション本体
├── requirements.txt     # Python依存パッケージ
├── templates/          # HTMLテンプレート
│   └── index.html
└── static/             # 静的ファイル（CSS, JavaScript）
    ├── styles.css
    └── script.js
```

## 🚀 セットアップ

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. アプリケーションの起動

```bash
uvicorn main:app --reload
```

または、特定のホスト・ポートで起動：

```bash
uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

### 3. ブラウザでアクセス

```
http://127.0.0.1:8000
```

## 🎯 機能

- **九星気学**: 生年月日から九星を算出し、2026年の運勢をスコアリング
- **四柱推命**: 年・月・日・時の四柱と五行バランスを表示
- **五星三心占い**: 独自のタイプ分類による性格診断
- **カバラ数秘術**: 運命数の算出と解説
- **宿曜占星術**: 27宿による性格・運勢・仕事・恋愛運
- **総合スコアリング**: 108通りの干支×九星の組み合わせをランキング化

## 📊 スコア計算方法

- **基準点**: 九星気学の2026年運勢スコア（5〜80点）
- **ボーナス**: 相生関係、月命星の影響など
- **合計**: 最大100点満点で評価

## 🔧 技術スタック

- **Backend**: FastAPI + Python
- **Frontend**: HTML5 + CSS3 + Vanilla JavaScript
- **Chart**: Chart.js（五行バランスのレーダーチャート）
- **占術計算**: 旧暦変換（lunardate）
